‘Midnight’은 Direct2D를 이용한 렌더링 방식으로 개발되었으며, <br>
컴포넌트 기반 엔진 구조 아키텍처를 기반으로 제작되었습니다.  <br>
주요 역할로는 파티클 시스템과 순찰자 AI 구현을 위한 FSM(유한 상태 기계) 개발을 담당하였으며, <br>
두 시스템 모두 하나의 컴포넌트로 구성되어 오브젝트에 장착되어 동작하도록 구현했습니다.<br>
